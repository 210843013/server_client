

#简单的时间客户端和服务器端



      client：
   
          1、创建一个IPv4(AF_INET)的字节流(SOCK_STREAM)套接字
          2、连接到IP地址为（192.168.1.108、127.0.0.1），端口号为8888的服务器
          3、写一个数据到服务器
          4、接收服务器传回来的数据并显示到标准输出
          5、关闭套接字
          
          
     server(提供时间服务的服务器端)：
    
          1、绑定众所周知的IPv4的地址（INADDR_ANY）和端口号（8888）
          2、将套接字转换成监听套接字
          3、睡眠等待客户端连接
          4、读取客户端数据
          5、将收到的数据发回给客户端
          6、睡眠1S后关闭套接字
